# MyStaticLib

01-03-2026

Denne folder indeholder blot et simpelt C++ class library, der er sat op til at skulle bygge under anvendelse af CMake.

Jeg fik hjælp af ChatGpt til at sætte det op og sagde, at jeg ville bruge VS Code, da jeg også vil kunne bygge det på en Linux platform.

Den foreslog at installere føglende:

* En compiler såsom Visual Studio 2026
* CMake
* VS Code
* Extensions i VS Code:
  * C++
  * C++ Extension Pack
  * CMake Tools
